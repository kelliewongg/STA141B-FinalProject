Kaggle links for our data, please get the csv files from the following links:<br>

US Accidents:<br>
https://www.kaggle.com/datasets/sobhanmoosavi/us-accidents<br>
Main dataset, includes the accidents from 2016-2023<br>

US Insurance:<br>
https://www.kaggle.com/datasets/andrewmvd/car-insurance-costs<br>
Includes the cost of car insurance for each US state<br>

US Population:<br>
https://www.kaggle.com/datasets/alexandrepetit881234/us-population-by-state<br>
Includes the population of each US State from the 2020 census<br>


The "Dealing with Car Accidents and Population.ipynb" file includes the code detailing the correlation between car accidents and population.<br>
The "foliummap.ipynb" file includes the code for the interactive map including a set seed sample of 50,000 accidents.<br>
The "seaborn_plots_accidents.ipynb" file includes the code plotting and solving for the correlation between car accidents and temperature, precipitation, wind speed, and humidity levels.
