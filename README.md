Kaggle links for our data, please get the csv files from the following links:

US Accidents:
https://www.kaggle.com/datasets/sobhanmoosavi/us-accidents
Main dataset, includes the accidents from 2016-2023

US Insurance:
https://www.kaggle.com/datasets/andrewmvd/car-insurance-costs
Includes the cost of car insurance for each US state

US Population: 
https://www.kaggle.com/datasets/alexandrepetit881234/us-population-by-state
Includes the population of each US State from the 2020 census
